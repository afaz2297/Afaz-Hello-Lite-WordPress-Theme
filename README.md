# Afaz Hello Lite

Afaz Hello Lite is a minimal, ultra-lightweight WordPress starter theme built for Elementor.  
Designed by **Afaz Alif**, this theme is fully responsive, Gutenberg-compatible, and WooCommerce-ready.  
Perfect for developers who want a clean canvas to build custom websites fast without unnecessary bloat.

---

## Features

- Minimal & lightweight
- Elementor-ready with Blank Page template
- Gutenberg block spacing ready
- Custom Logo support
- WooCommerce compatible
- Translation-ready
- Fully responsive layout

## Installation

1. Download or clone the repository.
2. Upload the folder `afaz-hello-lite` to your WordPress `/wp-content/themes/` directory.
3. Activate the theme via WordPress Admin → Appearance → Themes.
4. Use the page template **Blank Elementor** for full-width Elementor pages.
